# PyHeaders

Clang-based C/C++ header parser for python.

## Wanted Abilities

+ Get access to all compile time constants in your code
  * Print?
  * Access a specific one / a scope
+ Get string literals defined arbitrarily in the code (ie: log messages)
+ Detect constants that should be `constexpr` and flag them
+ Flag magic numbers and strings
+ Run from a docker
